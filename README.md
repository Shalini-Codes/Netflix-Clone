## 📌 Overview

A **Netflix-inspired UI clone** built using **HTML and CSS**.
This project replicates the look and feel of a modern streaming platform with a clean and responsive design.

---

## 🚀 Features

* 🎥 Netflix-style homepage
* 🔍 Search bar in navbar
* 🎞️ Multiple movie sections
* 🔐 Login & Signup pages
* 📱 Fully responsive design
* ✨ Smooth hover animations
